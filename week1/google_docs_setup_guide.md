# Week 1 Google Docs 워크샵 - 1분 셋업 가이드

## 🚀 수업 10분 전 체크리스트

### ✅ Step 1: Google Docs 새 파일 만들기 (20초)
1. Google Drive 열기
2. 새로 만들기 → Google 문서 클릭
3. 제목: "Week 1 Workshop - Human-Centered Writing (2025년 O월 O일)"

### ✅ Step 2: 템플릿 붙여넣기 (30초)
1. `google_docs_workshop_template.md` 파일 열기
2. 전체 복사 (Cmd/Ctrl + A → Cmd/Ctrl + C)
3. Google Docs에 붙여넣기 (Cmd/Ctrl + V)
4. ⚠️ 주의: 마크다운 문법이 그대로 붙여지는데 괜찮습니다!

### ✅ Step 3: 공유 설정 (15초)
1. 오른쪽 상단 "공유" 버튼 클릭
2. "링크가 있는 모든 사용자" 선택
3. 권한: "댓글 작성자" 선택 ← **중요!**
   - ✅ 학생들이 자기 섹션에 작성 가능
   - ✅ 다른 학생 섹션은 댓글만 가능
   - ❌ "편집자" 선택하면 전체 수정 가능 (혼란)
4. "링크 복사" 클릭

### ✅ Step 4: 링크 공유 (15초)
학생들에게 다음 메시지 발송:

```
Week 1 워크샵 Google Docs 링크입니다.

🔗 [붙여넣은 링크]

📌 준비사항:
- 자신의 초록 또는 서론 문단 (디지털 파일)
- 복사-붙여넣기 가능하게 준비

⏰ 수업 5분 전까지 접속 부탁드립니다.

💡 팁:
- 북마크 기능으로 본인 섹션 빠르게 찾기
- 댓글 기능으로 피드백 남기기
- 실시간 자동 저장됩니다
```

## 📝 수업 중 운영 팁

### Stage 1A: Bad Sentences (15분)

**[00:00-00:02] 시작 안내**
```
💬 교수자:

"모두 Google Docs에 접속되었나요?
오늘은 실시간으로 함께 작업합니다.

진행 방법:
1. 필수 3개 문장(#1, #5, #9) 작업
2. 본인 이름 옆에 직접 타이핑
3. 다른 학생들 수정안 보면서 학습
4. 댓글로 피드백

자, 'STAGE 1A'로 스크롤하세요. 시작!"
```

**[00:02-00:09] 개인 작업 모니터링**
- Google Docs 상단에 실시간 접속자 표시됨
- 각 학생이 어디서 작업 중인지 커서로 보임
- 막히는 학생에게 댓글로 힌트 제공
- 우수 답안에 즉시 댓글: "👍 정확합니다!"

**[00:09-00:15] 모범 답안 공개**
- "모범 답안" 섹션에 직접 타이핑
- 또는 미리 준비한 답안 붙여넣기

### Stage 1B: Bad Paragraphs (15분)

**진행 동일**:
- 학생들이 2개 문단 선택
- "학생___" 빈칸에 본인 이름 쓰고 작업
- 댓글로 피드백

### Stage 2: Smart Revising (20분)

**[00:30-00:32] 개인 작업 영역 찾기**
```
💬 교수자:

"이제 'STAGE 2'로 이동하세요.
Ctrl/Cmd + F로 '학생A'를 검색하면 빠릅니다.

본인 이름 섹션을 찾아
'내 초록' 영역에 초록을 붙여넣으세요.

그리고 7단계를 하나씩 체크하세요.
시작!"
```

**[00:32-00:42] 체크리스트 진행**
- 학생들이 각 Step을 순서대로 진행
- 교수자는 순회하며 댓글로 가이드
- 실시간으로 진행 상황 모니터링

**[00:42-00:50] 페어 피드백**
```
💬 교수자:

"페어를 확인하세요:
A↔B, C↔D, E↔F, G↔H, I↔J

페어의 섹션으로 이동해서
초록을 읽고 댓글로 피드백을 남기세요.

✅ 잘된 점 1개
💡 개선점 1개

5분 드립니다!"
```

## 🎯 Google Docs 특화 기능 활용

### 1. 목차 자동 생성
- 상단 메뉴: 삽입 → 목차
- 제목 서식(H1, H2)이 자동으로 목차에 표시
- 클릭하면 해당 섹션으로 이동

### 2. 북마크 기능
- 학생들이 본인 섹션에 북마크 설정
- 빠른 접근 가능

### 3. 댓글 기능
- 텍스트 선택 → 오른쪽 클릭 → 댓글
- @ 멘션으로 특정 학생 호출
- 해결됨 표시로 피드백 관리

### 4. 제안 모드
- 오른쪽 상단 연필 아이콘 → "제안"
- 학생들이 수정안을 "제안"으로 작성
- 교수자가 승인/거부 가능

### 5. 버전 기록
- 파일 → 버전 기록 → 버전 기록 보기
- 수업 전/중/후 버전 비교 가능

## ⚠️ 예상 문제 & 해결

### 문제 1: 학생이 다른 학생 섹션을 수정함
**예방**:
- 수업 시작 시 명확히 안내: "본인 이름 섹션만 작업"
- 권한을 "댓글 작성자"로 설정하면 자동 제한

**해결**:
- Ctrl/Cmd + Z로 실행 취소
- 또는 버전 기록에서 복원

### 문제 2: Google Docs가 느려짐
**원인**:
- 10명이 동시 편집
- 파일이 너무 김 (예상: 50+ 페이지)

**해결**:
- 실제로는 괜찮습니다 (Google Docs는 100명도 가능)
- 만약 느리면: Stage별로 파일 분리
  - Stage 1 파일, Stage 2 파일 따로

### 문제 3: 학생이 접속 못 함
**원인**:
- 링크 권한 문제
- Google 계정 미로그인

**해결**:
- "링크가 있는 모든 사용자" 확인
- 학생에게 Google 로그인 요청
- 최악의 경우: 화면공유로 참여 (읽기만)

### 문제 4: 모범 답안을 학생들이 미리 봄
**예방**:
- 모범 답안 섹션을 처음엔 비워두기
- 학생 작업 후 실시간으로 작성

**또는**:
- 별도 파일에 준비
- 적절한 시점에 복사-붙여넣기
