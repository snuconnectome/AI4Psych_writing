# Week 1 워크샵 - 초간단 시작 가이드 🚀

## 🎯 이게 뭔가요?

Google Docs로 진행하는 **실시간 협업 워크샵** 템플릿입니다.
- ✅ 10명 학생 동시 작업
- ✅ 실시간 피드백
- ✅ 1분 셋업
- ✅ Figma 필요 없음

---

## ⚡ 1분 시작 방법

### 1️⃣ Google Docs 새 파일 만들기
- Google Drive → 새로 만들기 → Google 문서

### 2️⃣ 템플릿 붙여넣기
- `google_docs_workshop_template.md` 파일 열기
- 전체 복사 → Google Docs에 붙여넣기

### 3️⃣ 링크 공유
- 오른쪽 상단 "공유" 클릭
- 권한: **"댓글 작성자"** ← 중요!
- 링크 복사 → 학생들에게 발송

### 4️⃣ 수업 시작!
- 학생들이 접속하면 바로 시작

---

## 📁 파일 구조

### `google_docs_workshop_template.md` ← 메인 파일
워크샵 전체 내용 (복사해서 Google Docs에 붙여넣기):
- Stage 1A: Bad Sentences (10개 문장)
- Stage 1B: Bad Paragraphs (4개 실제 논문 문단)
- Stage 2: Smart Revising (10명 개인 작업 구역)

**길이**: 약 1,500 줄 (Google Docs에서 약 40-50 페이지)

### `google_docs_setup_guide.md` ← 상세 가이드
교수자를 위한 운영 가이드:
- 수업 전 체크리스트
- 수업 중 진행 스크립트
- 문제 해결 방법
- 학생 가이드 (1-page)

---

## 🎬 수업 흐름 (50분)

### Stage 1A: Bad Sentences (15분)
- 필수 3개 (#1, #5, #9) + 선택 2개
- 10명이 각자 문제점 찾고 수정
- 댓글로 피드백 교환
- 교수자가 모범 답안 공개

### Stage 1B: Bad Paragraphs (15분)
- 실제 PLOS ONE 논문 4개 문단
- 각자 2개 선택하여 개선
- Peer review
- 모범 답안 공개

### Stage 2: Smart Revising (20분)
- 각자 본인 초록을 7단계 체크리스트로 점검
- 페어와 피드백 교환 (A↔B, C↔D, ...)
- 최종 수정 버전 작성

## 💡 핵심 장점

### vs Figma
| 기능 | Google Docs | Figma |
|------|-------------|-------|
| 셋업 시간 | **1분** | 30-60분 |
| 학습 곡선 | **즉시 사용** | 5-10분 설명 필요 |
| 실시간 협업 | ✅ | ✅ |
| 댓글 기능 | ✅ | ✅ |
| 학생 친숙도 | **매우 높음** | 낮음 |
| 텍스트 작업 | **최적화됨** | 불편함 |

### 실시간 협업
- 10명의 커서가 실시간으로 보임
- 누가 어디서 작업 중인지 확인 가능
- 즉각적인 댓글 피드백
- 자동 저장 (버전 기록 보존)

### 간편한 관리
- 별도 소프트웨어 설치 불필요
- 브라우저만 있으면 됨
- 모든 기기에서 접속 가능
- 수업 후 PDF 저장 가능

## 🆘 문제 해결

### "학생이 다른 학생 섹션을 수정했어요"
→ Ctrl/Cmd + Z로 실행 취소
→ 또는 버전 기록에서 복원
→ 예방: 권한을 "댓글 작성자"로

### "Google Docs가 느려요"
→ 실제로는 괜찮습니다 (10명 정도는 문제없음)
→ 만약 느리면: Stage별로 파일 분리

### "학생이 접속 못 해요"
→ 링크 권한 확인: "링크가 있는 모든 사용자"
→ Google 계정 로그인 확인
→ 최악: 화면공유로 참여
