# 📚 Week 1 원칙 참조 (항상 참고하세요)

## Lesson 1: 주어-동사

### ✓ 동작을 동사에 담기
- ❌ performed an analysis → ✅ analyzed
- ❌ made an improvement → ✅ improved
- ❌ conducted an investigation → ✅ investigated

### ✓ 주체를 주어에 두기
- ❌ The data shows... → ✅ We found...
- ❌ It was observed that... → ✅ We observed that...

### ✓ 주어-동사 가깝게 (10 words 이내)
- ❌ The study [12 words...] found → ✅ The study found... [then details]

---

## Lesson 2: 응집성

### ✓ Old → New 정보 흐름
- 각 문장은 이전 문장의 끝 정보로 시작
- 새 정보는 문장 끝에 배치

### ✓ 전략적 수동태
- Old→New 흐름에 필요할 때만 사용
- 주체가 불명확할 때는 능동태 선호

### ✓ 문단 응집성
- 첫 문장과 마지막 문장이 같은 주제
- 모든 문장이 문단의 주제를 지지

---

## Lesson 3: 간결성

### ✓ 불필요한 단어 제거
- "it should be noted that" → 삭제
- "due to the fact that" → "because"
- "a large number of" → "many"

### ✓ 간단한 언어
- "utilize" → "use"
- "methodology" → "method"
- "eudemonic" → "fulfilling"

### ✓ 수식어 제한
- "very", "really", "quite" → 대부분 삭제 가능

## 📝 Bad Sentences #2-4, #6-8, #10 (선택 - 각자 2개 선택)

**시간이 있으면 추가로 연습하세요. 아래 문장 중 2개를 골라 같은 방식으로 작업:**

### #2: 주체 위치 문제
```
Using sarkosyl to induce nuclear run-on, the transcriptionally inactive
b-globin gene in mature erythrocytes was demonstrated to harbor high levels
of Pol II at 5′ proximal regulatory regions.
```

### #3: 주어-동사 거리
```
Peanuts, shrimp, almonds, milk or anything else with lactose, and wheat or
anything with gluten all represent things that people are commonly allergic to.
```

### #4: Nominalization + 수동태
```
Mapping of open chromatin regions, post-translational histone modifications
and DNA methylation across a whole genome is now feasible, and new non-coding
RNAs can be sensitively identified via RNA sequencing.
```

### #6: 수동태 모호성
```
Improvements are expected in the predictive power of all the scores being
computed on multispecies alignments.
```

### #7: 응집성 문제 (2문장)
```
We identified genes that are differentially expressed between species.

A phylogenetic tree based on the number of differentially expressed genes
between species recapitulates their known phylogeny.
```

### #8: 불필요한 단어
```
These approaches use different kinds of methodology.
```

### #10: 과도한 Hedging
```
We estimated that as much as 12-18% (depending on the tissue) of the total
variance in gene expression might be explained, at least in part, by changes
in DNA methylation patterns.
```
